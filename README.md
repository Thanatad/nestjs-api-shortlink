## Description

A ShortLink application
* TypeORM

## Installation

```bash
$ cp .env.example .env.development ; cp .env.example .env.production
$ pnpm install
```
## Running the app

```bash
# development
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
